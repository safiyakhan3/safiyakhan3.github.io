About This Project

This site is a class project to practice semantic HTML, clean CSS, and basic accessibility. Content emphasizes clarity and meaningful structure: headings, paragraphs, lists, and images with descriptive alternative text.

Key Terms

HTML
Gives structure and meaning to web content through semantic elements.
CSS
Controls the presentation—fonts, colors, layout, spacing—separate from content.
Alt Text
Short descriptions of images that make visual content accessible to screen-reader users.
